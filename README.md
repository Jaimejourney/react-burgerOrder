# react-burgerOrder

一个基于reactJs,react-redux和react-thunk的single-page app,用于DIY你的汉堡，填写订单和客户信息并且结账，简单的页面：

![image-20190306164647400](/Users/yikaizhu/Library/Application Support/typora-user-images/image-20190306164647400.png)

在终端执行(安装package.json中列出来的依赖)：

```
npm install
```

运行：

```
npm start
```

然后在多个浏览器中访问: http://localhost:9000, 就可以选购你的汉堡🍔了
